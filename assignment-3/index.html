Assignment 3 - Maps



Tree Abundance per Neighborhood

The first map of the Trento's circoscrizioni shows the different tree abundance
(total number of trees) inside each region.
The increasing of the intensity of green scale color follows the increasing of the tree abundance.

Map of Trento's circoscrizioni



Tree Density per Neighborhood

The second map of the Trento's circoscrizioni shows instead the different tree density
(total canopy cover of a region divided by the area of the same region) inside each region.
The increasing of the intensity of green scale color follows the increasing of the tree density.

Map of Trento's circoscrizioni



Oxygen Production per Neighborhood

In this third map of the Trento's circoscrizioni we are able to see the different tree oxygen production
(measured in kg per year) inside each region.
The increasing of the intensity of blue scale color follows the increasing of the tree oxygen production.
