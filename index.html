EcoBenefits Project

Pesto Team (Nabizada, Afghan and Ismail)

The final goal of this project is to create visualizations that highlight the ecological advantages provided by trees in Trento. This involves performing calculations and linking the "price" of trees to these advantages. By presenting the data in a visually appealing and understandable way, the website aims to communicate the importance and benefits of trees in the municipality of Trento.

To achieve this, the PESTO! group has utilized various techniques of data visualization, leveraging the data specifically focused on trees' abundance and position in Trento. The dataset itself consists of JSON and GeoJSON files, containing information about different tree attributes such as species, height, circumference, crown breadth, and geographic coordinates.

By utilizing HTML, CSS, and the D3.js library, the group has created charts and visualizations that present the data in an accessible manner. The website provides subpages dedicated to different assignments, each showcasing different visualizations and techniques to convey conclusions drawn from the Trento tree dataset.

Data visualization for trento tree dataset

For the initial task, we need to analyze and compare the abundance and distribution of different tree species across various "circoscrizioni" (neighborhoods) where the trees are located. Our approach involved creating five distinct charts:

The first chart presents a straightforward barchart displaying the top 20 tree species based on their abundance, along with additional information. The second chart employs a stacked barchart to compare the relationship between neighborhoods and the top 5 tree species.

Similar to the previous chart, the third one serves the same purpose but adopts small multiple charts, each representing a specific tree species. The fourth chart is a modified version of the second one, displaying a stacked bar chart that showcases the percentage distribution of tree species.

Lastly, the fifth chart utilizes waffle charts to illustrate the percentage distribution of tree species across neighborhoods.
